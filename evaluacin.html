Pregunta

1.¿Qué es un sistema operativo?

Respuestas

 a) Un componente físico que permite a la computadora funcionar

b) Una serie de programas organizados para gestionar hardware, aplicaciones y usuarios

 c) Un programa diseñado exclusivamente para editar textos.

d) Una interfaz gráfica que reemplaza al hardware.

Retroalimentación

Incorrecto

Opción correcta

Incorrecto

Respuesta correcta: Literal b)

Un sistema operativo es una serie de programas organizados para gestionar hardware, aplicaciones y usuarios.

Solución

  1. Incorrecto (Retroalimentación)
  2. Opción correcta (Retroalimentación)
  3. Incorrecto (Retroalimentación)
  4. Incorrecto (Retroalimentación)
